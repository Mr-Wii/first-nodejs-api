**CRUD NodeJs Rest API (JWT)**

---

- Register :

       /api/auth/register

* Get whole user object (after getting & providing the token) :

       /api/auth/me

- Get the users :

       /api/user

- Get user by ID :

       /api/user/ :id

- update user by ID :
    
   /api/user/ :id

- delete user by ID :

      /api/user/ :id
